## 🔧 技术修复

### 问题 1：页面切换时卡片抖动

**原因**: `bento-fade-init` 类中的 `opacity: 0` 导致初始渲染闪烁，结合 transform 动画在切换时触发重排。

**解决方案**:
- 移除级联动画的 `transform: translateY(16px)` 初始值
- 仅保留 `opacity: 0 → opacity: 1` 的平滑过渡
- 去除 `will-change: transform`，改用渲染优化

**修改文件**: `src/styles/bento-ui.css`

```css
/* 前 - 导致抖动 */
.bento-fade-init {
  opacity: 0;
  transform: translateY(16px);  /* ❌ 导致重排 */
}

/* 后 - 仅淡入，无位移 */
.bento-fade-init {
  opacity: 0;  /* ✅ 仅透明度变化 */
}

.bento-fade-in {
  opacity: 1 !important;
  transition: opacity 0.5s cubic-bezier(0.22, 1, 0.36, 1) !important;
}
```

---

### 问题 2：卡片顶部蓝/橙色细条

**原因**: CSS 伪元素 `::before` 为工具卡片添加蓝色、游戏卡片添加橙色的顶部 3px 细条，视觉上显得不够简洁。

**解决方案**:
- 完全移除 `a.group[href*="/tools/"]::before` 和 `a.group[href*="/games/"]::before` 伪元素规则
- 改用单纯的边框和阴影来传达卡片层级

**修改文件**: `src/styles/bento-ui.css`（第 91-117 行删除）
