# UI 设计与交互优化记录

## 1. 文案优化 (去 AI 化)
为了让网站显得更真实、更贴近用户，我们对全站的文案进行了优化，移除了过度使用的“AI”字眼，替换为更自然、更具科技感或更符合场景的词汇。

### 首页 (`src/pages/Home.jsx`)
- `AI对话导航插件` -> `智能对话导航插件`
- `AI桌游聚会游戏` -> `欢乐桌游聚会游戏`
- `用AI，让每件事都更简单` -> `用科技，让每件事都更简单`
- `精选AI效率工具...` -> `精选效率工具...`
- 标签中的 `AI` 替换为 `智能`

### 游戏广场 (`src/pages/Games.jsx`)
- `AI桌游聚会游戏` -> `欢乐桌游聚会游戏`
- `AI剧本杀` -> `沉浸剧本杀`
- `AI知识竞答` -> `趣味知识竞答`
- `AI 驱动` -> `智能 驱动`
- `🤖 AI 实时生成` -> `🤖 智能实时生成`
- 描述中的 `AI出题`、`AI生成` 等替换为 `智能出题`、`智能生成`

### 工具库 (`src/pages/Tools.jsx`)
- `AI工具库` -> `精选工具库`

### 关于我们 (`src/pages/About.jsx`)
- 路线图中的 `AI对话导航插件`、`AI聚会游戏` 等进行了相应替换。
- 站长介绍中的 `喜欢用AI解决生活中的小问题` -> `喜欢用科技解决生活中的小问题`
- 底部版权 `用 ❤️ 和 AI 打造` -> `用 ❤️ 和 科技 打造`

### 聚会游戏详情页 (`src/pages/PartyGame.jsx`)

## 2. 交互逻辑优化
在不影响底层功能的前提下，增加了一些平滑的过渡动画，提升用户体验。

- **首页工具卡片**：添加了 `animate-in fade-in slide-in-from-bottom-4` 动画，使卡片在加载时有从底部淡入的效果。
- **工具库卡片**：同样添加了淡入和上滑的动画效果。
- **游戏广场卡片**：为游戏卡片添加了淡入和上滑的动画效果。
- **工具详情页**：底部的“相关工具推荐”卡片添加了淡入和上滑的动画效果。
- **关于我们**：路线图列表项添加了带有延迟的级联淡入动画 (`animationDelay`)，使列表项依次出现，视觉效果更丰富。
